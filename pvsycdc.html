Es una aplicación para “Windows”, ejecutable en cualquier sistema operativo de ventanas de Microsoft cuya versión sea superior o igual a la 4 (Windows 95, Windows NT 4.0, Windows 98, Windows Me, Windows 2000 y posteriores).

Dispone de un sistema de instalación automático. También se puede desinstalar automáticamente desde el Panel de Control de Windows (Agregar o quitar programas).

Está realizada para llevar el control de gastos de combustible y seguros de una empresa que tenga una flota de vehículos. Controla el gasto de combustible por vehículo, los vencimientos de seguros.

Se pueden realizar filtros, búsquedas, gráficos de decisión, ...

Precio: 90 € (precio orientativo, variará en función del volumen de información a tratar)
