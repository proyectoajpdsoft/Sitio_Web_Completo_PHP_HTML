Es una aplicación para Windows, ejecutable en cualquier sistema operativo de Microsoft cuya versión sea superior o igual a la 4 (Windows 95, Windows NT 4.0, Windows 98, Windows Me, Windows 2000 y Windows XP).

Dispone de un sistema de instalación automático. También se puede desinstalar automáticamente desde el Panel de Control de Windows (Agregar o quitar programas).

Este programa está dirigido para autónomos o particulares (jardineros, abogados, mecánicos...) que trabajen por su cuenta.

Características:

  • Facturación, presupuestos, diario de ingresos, cobros, gastos, clientes, acreedores. Generación de facturas mensuales por grupos de clientes, generación de ingresos mensuales por grupos de clientes.
  • Impresión de todos los datos (facturas, acreedores, clientes, presupuestos, gastos, diario, ...)
  • Exportación de datos a HTML, Excel, Texto, ...
  • ...

 

Precio: 90 € (precio orientativo, variará en función del volumen de información a tratar)
