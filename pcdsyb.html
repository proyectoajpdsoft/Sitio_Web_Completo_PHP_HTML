Es una aplicación para Windows, ejecutable en cualquier sistema operativo de Microsoft cuya versión sea superior o igual a la 4 (Windows 95, Windows NT 4.0, Windows 98, Windows Me, Windows 2000 y Windows XP).

Dispone de un sistema de instalación automático. También se puede desinstalar automáticamente desde el Panel de Control de Windows (Agregar o quitar programas).

Está realizada para calcular el número de billetes y monedas de euro necesarias para realizar el pago en efectivo a los obreros. A partir de la introducción de los partes de trabajo diarios de los obreros (agrupados por cuadrillas) el programa genera la cantidad de billetes y monedas que hay que pagar, por obrero y en total.

Se pueden realizar, a su vez, cargas de datos de Microsoft Access, impresión de obreros, cuadrillas, pagos, ...

 

Precio: 270 € (precio orientativo, variará en función del volumen de información a tratar)
