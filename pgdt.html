Gestión de transportes Descargar versión de evaluación

Tamaño: 9,27 MB
Es una aplicación para “Windows”, ejecutable en cualquier sistema operativo de ventanas de Microsoft cuya versión sea superior o igual a la 4 (Windows 95, Windows NT 4.0, Windows 98, Windows Me, Windows 2000 y posteriores).

Dispone de un sistema de instalación automático. También se puede desinstalar automáticamente desde el Panel de Control de Windows (Agregar o quitar programas).

Está realizada para llevar el control de una empresa de transportes.


En cualquier momento se puede saber:

· Gastos, ingresos, beneficio por camión.

· Facturación, portes, reportes, viajes, clientes, camioneros, camiones.

· Seguros, gastos de camiones, gráficos de decisión, totales mensuales, totales anuales (kilómetros, días de viaje, beneficio, gastos, ingresos, combustible, autopistas, tasas, consumo de combustible, ...).

· Kilómetros realizados por camionero, viajes realizados por camionero, total beneficio camionero.

· ...

 

Precio: 660 € (precio orientativo, variará en función del volumen de información a tratar)
