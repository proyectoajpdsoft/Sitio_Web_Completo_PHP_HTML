Gestión de productos fitosanitarios y nutricionales Descargar versión de evaluación aún no disponible

Tamaño: no disponible
Es una aplicación para “Windows”, ejecutable en cualquier sistema operativo de ventanas de Microsoft cuya versión sea superior o igual a la 4 (Windows 95, Windows NT 4.0, Windows 98, Windows Me, Windows 2000 y posteriores).

Dispone de un sistema de instalación automático. También se puede desinstalar automáticamente desde el Panel de Control de Windows (Agregar o quitar programas).

Está realizada para llevar el control de una empresa agrícola. Incorpora control de stock de almacén, productos fitosanitarios y nutricionales (precios, materias activas, plazo de seguridad, distribuidores, formulación, envases, ...).

Incluye también control de albaranes de productos (entradas de productos), control de salidas de productos a fincas, devoluciones, ...

Permite saber en cualquier momento el stock disponible de productos, así como una lista de precios actualizada (mediante importación de la lista de productos de los distribuidores en formato ACCESS). Posibilidad de buscar productos por materias activas y formulación y viceversa.

Precio: 570 € (precio orientativo, variará en función del volumen de información a tratar)
