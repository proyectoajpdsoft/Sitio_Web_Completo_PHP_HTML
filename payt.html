Abonados y tratamientos Descargar versión de evaluación aún no disponible

Tamaño: no disponible
Es una aplicación para Windows, ejecutable en cualquier sistema operativo de Microsoft cuya versión sea superior o igual a la 4 (Windows 95, Windows NT 4.0, Windows 98, Windows Me, Windows 2000 y Windows XP).

Dispone de un sistema de instalación automático. También se puede desinstalar automáticamente desde el Panel de Control de Windows (Agregar o quitar programas).

Está realizada para gestionar los planes de abonados (para nutrición) y los planes de tratamientos (aéreos). Calcula e imprime los productos necesarios por período de riego. Permite generar los planes para varias fincas/propietarios.

Puede calcular los totales de productos consumidos por plan de abonado/plan de tratamiento así como por finca/propietario. También calcula los metros cúbicos de agua consumida por propietario y finca.

 

Precio: 270 € (precio orientativo, variará en función del volumen de información a tratar)
