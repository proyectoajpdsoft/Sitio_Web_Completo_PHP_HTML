Cómo realizar actualizaciones automáticas de los programas vía Web

Sólo podrán realizar actualizaciones automáticas los usuarios de pago que estén registrados. Para registrarse si aún no lo ha hecho pulse aquí. Una vez registrado, deberá esperar a que le confirmemos su registro.

Cuando haya realizado estos pasos, inicie su sesión con su código de usuario y su clave pulsando aquí, a continuación deberá descargar a su equipo el fichero "actualizar.exe", pulsando en el enlace: "Descargar fichero ejecutable" que le aparecerá al iniciar la sesión. Descargue también el fichero "configuracion.ini" pulsando en el enlace "Descargar fichero de configuración", este fichero debe copiarlo (o descargarlo) en la misma carpeta que el fichero "configuracion.exe" (descargado anteriormente).

Nota: el fichero de configuración ("configuracion.ini") será diferente para cada aplicación que tenga registrada con nosotros. Si tiene varias aplicaciones deberá tener una carpeta de actualización para cada una de ellas con el fichero "actualizar.exe" y el fichero "configuracion.ini" (de la aplicación que  corresponda).
